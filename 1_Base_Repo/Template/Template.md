Test type : Manual
Test case Name: All Work Items View - UI
Description: Display all active work item for the logged in user
Actions:1. Navigate to the All Work Items section
Observe the list of work items displayed on the screen.
Check the status of each work item to ensure only active items are shown.
Verify the ownership of each item to confirm they belong to the logged-in user.
The screen displays a list of work items where Case Status is marked as Processed, as expected.
verify system should navigate to the Assign Me To screen.
verify screen should display a list of all work items assigned to the logged-in user.
verify system should navigate to the Processed work item screen.
The screen should display a list of all work items where Case Status = Processed.
Verify system should navigate the unmatched tab
verify system displays the required work item clearly.
verify checkbox is available next to the work item.
Once the checkbox is marked, the "Start Claim" button becomes active.
Clicking the "Start Claim" button initiates the claim process successfully.
Expected Results:Successfully navigated to All work items
A list of work items should be displayed
Only active work items are listed
All displayed items are assigned to the currently logged-in user.
Work items are categorized into To Do, In Progress, and Processed.
The Assign Me To tab is present and clickable.
The screen should displays the list of work items assigned to the logged-in user, as expected.
Processed work item screen should be present and clickable
The screen displays a list of work items where Case Status is marked as Processed, as expected.
The unmatched tab should be present and clickable
The work item is visible and correctly listed.
The checkbox is present and can be selected.
After marking the checkbox, the "Start Claim" button is enabled.
The claim process starts without any errors upon clicking the button.
